# 1. Gameplay

## 1.1. Auto-Climb during Sprint

Set to: **No**

Utility: disables **autoclimb**, particularly for stratagem's **pod panel**, **resupply**, certain **textures**, etc.

![](/static/blob/pzm7kvkz.png)

_____

# 2. Display

## 2.1. Camera Shake Strength

Set to: **Off**

Utility: reduces **camera shake** caused by **gunfire** and **explosions**. Useful for better **aiming** and **stability** (especially when using the **Anti-Tank Emplacement**).

![](/static/blob/qny0689e.png)

_____

## 2.2. Vertical Field of View

Set to: **90**

Utility: offers a **wider field of view**. Useful for spotting **enemies** approaching from the **sides** or **behind**.

![](/static/blob/7z33gvgz.png)

_____

## 2.3. Framerate Limit

Set to: **On** then limit to **60**

Utility: prevents the **first pod spawn** to be almost instantly on the ground when you **start** a mission.

![](/static/blob/dz4jp5dz.png)

_____

# 3. Graphics

## 3.1. Vegetation and Rubble Density

Set to: **Low**

Utility: reduces the **thickness** of **bushes**, improving your **mobility**.

![](/static/blob/1n8j0xqe.png)

_____

## 3.2. Volumetric Fog Quality

Set to: **Lowest**

Utility: reduces the **thickness** of **environmental fog**, improving your **visibility**.

![](/static/blob/3nv2d6vz.png)
