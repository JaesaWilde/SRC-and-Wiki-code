This guide covers how to use the [TimeDivers manager](https://github.com/leem919/timedivers-manager/) created by **leem919** to **downpatch** Helldivers 2. This method is generally more suitable for non-technical users and users new to downpatching.

Other resources:

- [How to play Meridia/Gloom missions](https://www.speedrun.com/Helldivers2/guides/ep9oo)
- [Downpatch your game (manually)](https://www.speedrun.com/Helldivers2/guides/zf9n8)

**WARNING**: if you are downpatching to a version prior to **Omens of Tyranny**, ensure that you are **not** on an **Illuminate planet** before changing versions, otherwise your game will crash. It **won't crash** if you launch the game with the [DarkFluid Launcher](https://github.com/WernherVonKerman/DarkfluidLauncher/releases).

_____

# 1. On Steam

## 1.1. Updates

Navigate to the game in your Steam library. Make sure it's **up to date**. Right click it, then select ***Properties***.

![](/static/blob/qny0344e.png)

_____

Click on **Updates**, then set ***Automatic Updates*** to ***Wait until I launch the game***.

![](/static/blob/3nv2q0dz.png)

_____

## 1.2. DLCs

Click on ***DLC***, then ensure everything is **unchecked**.

![](/static/blob/1n8jo69e.png)

_____

# 2. Get the version manager

## 2.1. Download the version manager

Navigate to the [releases section](https://github.com/leem919/timedivers-manager/releases) of the **TimeDivers Manager**. Download the **latest version** of ***mainfests.json*** and ***timediversverman.exe***.

![](/static/blob/rz2426on.png)

_____

Create a **new folder** for the files you just downloaded. It **doesn't matter** what you call it. For the example, we will name it **TimeDivers**.

![](/static/blob/5e17356e.png) &nbsp; ![](/static/blob/oeoxk7ke.png)

_____

## 2.2. Download the Steam Depot Downloader

Navigate to the [releases section](https://github.com/SteamRE/DepotDownloader/releases) of the **Steam Depot Downloader**. Download the **latest version** for your PC. For most **Windows** users, this will be ***DepotDownloader-windows-x64.zip***.

![](/static/blob/pzm71j4z.png)

_____

Open the **zip file** and extract ***DepotDownloader.exe*** to the **same folder** you placed ***mainfests.json*** and ***timediversverman.exe*** in.

![](/static/blob/xz0jl3me.png)

_____

Now your **folder** should look like this:

![](/static/blob/kzwm9q2n.png)

_____

# 3. Setup the version manager

## 3.1. Path to Steam

Click on ***timediversverman.exe*** to launch the manager. Then click ***Browse*** to set the **location** of your **Steam game files**.

![](/static/blob/1n5jg8qe.png)

_____

By **default** for most users, this can be found in the following **path**:

![](/static/blob/1n7v1q7n.png)

_____

Once you have navigated there, click ***Select Folder***.

![](/static/blob/7n6vp84z.png)

_____

## 3.2. Sign in

Enter your **Steam username** that you use to sign in to the account. Check the box for ***Remember Password for Downloads***.

![](/static/blob/rnqvwx2e.png)

_____

## 3.3. Select a version

Scroll down the list until you find the **version** you need. For this example, I will choose **1.003.000** as it is the **most common** patch for downpatched runs at the moment. I select it and click Click ***Download Version***.

![](/static/blob/7e93k63e.png)

_____

A **Windows command prompt** will open and begin downloading the version you requested *via* the **Depot Downloader**. If prompted, enter your **Steam password** into the prompt, and **Steam Guard code**. This is required so Steam knows you **own** the game.
_____

## 3.4. Swap in, swap back

Once downloaded, ensure the game is **closed**, select the **patch**, then select ***Set Active Version*** to **swap** to the patch. You can now the launch the game on downpatch!

![](/static/blob/ynl9p10n.png)

_____

To **swap back** to **current patch**, ensure the game is **closed** and select ***Steam Version***. Then click ***Set Active Version***.

![](/static/blob/7z33m74z.png)
_____

# 4. Useful patches for speedrunning

Each patch costs between **75** and **135 gigas**.

**Release**: June 1st 2024.
- Infinite Grenade glitch.
- Last patch for the original Meridia missions event.
- Last patch where Retrieve Essential Personnel is playable.
- Stratagem replacement exploit to equip both Patriot and Emancipator Exosuits.

**Good for**:
- Meridia missions (NMG).
- Retrieve Essential Personnel (NMG Â· Unrestricted).

## 1.001.100
**Release**: September 17th 2024.
- Invincible Shield Pack glitch.
- Emergency Evacuation skip.

**Good for**:
- Solo Emergency Evacuation (Unrestricted).
_____

## 1.001.104
**Release**: October 15th 2024.
- Emergency Evacuation skip.
- Last patch where Meridia missions are playable.

**Good for**:
- Emergency Evacuation (Unrestricted).
- Meridia missions (Unrestricted).
_____

## 1.002.203
**Release**: April 4th 2025.
- Old Galactic Map UI.
- Pacifist skip.
- Pre-buffed bulk fabricators.
- Pre-nerfed Ultimatum.

**Good for**:
- Un-Scouted Automaton operations (Unrestricted).
_____

## 1.003.000
**Release**: May 13th 2025.
- First patch with Speedy Pelican (lastly effective on **1.003.105**).
- Flag of Super Earth planting to carry packages/shells glitch.
- Illuminate Megacities added.
- Pre-nerfed Repel Invasion Fleet.
- Stratagem Duping glitch.
- Stratagem Duplication with Weapon Customization glitch.

**Good for**:
- Any Illuminate/Automaton mission/operation (Unrestricted).
- Any non-Megacity Terminid mission (Unrestricted).
_____

## 1.003.101
**Release**: June 10th 2025.

- Ability to stim mid-dive.

- Any Megacity Terminid mission (Unrestricted).
