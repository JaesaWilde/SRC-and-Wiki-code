# 1. Download HD2ModManager and HD2 Cinematic Tool

- Register to [Nexus Mods](https://www.nexusmods.com/).
- Download and install the [HD2ModManager](https://www.nexusmods.com/helldivers2/mods/109?tab=files) · **Main files** > **Manual download**.
- Download the [HD2 Cinematic Tool](https://www.nexusmods.com/helldivers2/mods/593?tab=files) · **Main files** > **Manual download** and **Optional files** > **Manual download**.

To install the **speedrun addon**, you need to install the **Cinematic Tool** mod first. To do so:

- Unzip it and read the **README.txt**.
- Once you have it installed, unzip the **speedrun addon** and follow the instructions of the **README.png** file.

# 2. How to use it

While you're in the **loading screen**, press **F7** (default keybind). It should remove **background effects** from the **chat box**, **tip box**, **player info box** as well as the **black stripes**:

![](/static/blob/rnqvy52e.jpg)

The **fade-in** will not occur and the very first **in-game frame** will look like the image below.

**WARNING**: it's mandatory to **turn it off** by pressing **F7** once more, for instance when you **Helldiver exits their hellpod**. Otherwise, it will hide **HUD elements** whose visibility is **mandatory** for verification.

![](/static/blob/7e93pg3e.jpg)
