# Meridia missions

To submit a run of the following missions to the **100% No Major Glitches** category, you **must** play on patch **1.000.305**, use **none of the glitches** found in this version and submit **all POVs**. You can submit a run from any patch to the **100% Unrestricted** subcategory and use any glitch/exploit/skip.
