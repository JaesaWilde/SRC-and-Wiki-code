# Hive World missions

The following **Hive World** missions feature a **hidden optional objective**: Kill the Hive Lord. Since the game grants **full XP** without killing it, you may submit a run to the **No Hive Lord** subcategory for both:
- Runs where it spawns, and you have full cleared the map **without killing it**,
- Runs where it does not appear.
